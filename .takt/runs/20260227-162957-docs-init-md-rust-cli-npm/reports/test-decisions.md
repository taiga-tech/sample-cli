# 決定ログ

## 1. プランの6ケースに加えエッジケース2件を追加
- **背景**: タスク指示書は6ケースを列挙しているが、境界値・エッジケース網羅のポリシーに従い追加検討が必要だった
- **検討した選択肢**:
  - プランの6ケースのみ実装
  - エッジケース（darwin + isGlibc=true、未対応 arch）を追加
- **理由**: 2件の追加が有意義な境界値テストとして機能するため採用
  1. `darwin/arm64 + isGlibc=true` — `isGlibc` フラグが linux 以外で無視されることを明示的に検証
  2. `linux/arm64 (未対応 arch)` — platform が存在するが arch が未対応の場合のエラーメッセージ形式を検証
