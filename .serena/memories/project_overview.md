sample-cli is a monorepo for distributing a Rust native CLI through npm. Rust code lives in crates/core and crates/cli. The npm launcher lives in packages/cli and resolves platform-specific packages packages/cli-<platform>, which contain vendored binaries under vendor/<target-triple>/sample-cli/. There are also Next.js apps in apps/web and apps/docs, but they are not required for core CLI execution. Supporting scripts include scripts/place-binary-like-ci.sh for placing binaries in the same layout as CI.
